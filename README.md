# Orbit UI

AI-chat-first Orbit product UI prototype built with React, TypeScript, Vite, and Lucide icons.

## Features

- Orbit AI as the home entry.
- Contextual right-side result panels for event recommendations, people recommendations, and network analysis.
- Animated relationship orbit with the user avatar at the center and contacts orbiting around it.
- Dark and bright themes.
- English, Chinese, and Japanese UI copy.
- Mock data for events, people, cards, and analytics.

## Commands

```bash
npm install
npm run dev
npm test
npm run build
```

The local dev server defaults to `http://127.0.0.1:5173/`.
